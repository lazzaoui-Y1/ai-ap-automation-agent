# ai-ap-automation-agent
An intelligent AI-powered Invoice Processing Agent that automates extraction of structured data from PDF/image invoices (Arabic &amp; French &amp; English) using OCR and Groq's Llama-3.3-70B LLM. Features multi-customer isolation, pluggable ERP connectors ( Excel, CSV, SAP, SFTP, Oracle, Webhook) , FastAPI backend, and easy onboarding for new clients.
